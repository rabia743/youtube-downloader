📥 YouTube Video Downloader (Python)

A lightweight Python-based tool that allows users to download YouTube videos in the highest available quality using yt-dlp.

🚀 Key Features
Downloads videos in best available quality
Automatically saves files using the original video title
Fast, lightweight, and efficient
Simple and beginner-friendly implementation
✅ Requirements

Install the required dependency before running the script:

pip install yt-dlp

▶️ Usage Instructions
Run the Python script
Enter the YouTube video URL when prompted
Press Enter
The download will start automatically

📂 Output Location

Downloaded videos are saved in the same directory where the script is executed, using the video’s title as the filename.

💡 Advantages
Eliminates manual downloading steps
Saves time and effort
Requires minimal setup
Easily extendable for advanced features (audio extraction, playlists, GUI, etc.)
